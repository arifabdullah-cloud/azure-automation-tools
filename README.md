# Azure Automation Tools

This repository contains practical cloud automation utilities built for Microsoft Azure environments.

It focuses on cost optimization, monitoring, and infrastructure automation using Python and Azure SDKs.

---

## 🔧 Modules

### Cost Management
Tools for identifying and managing idle or underutilized resources.

### Monitoring
Scripts for collecting metrics and detecting system anomalies.

### Automation
Operational scripts for cloud task automation.

### Core
Shared utilities and reusable components.

### Infrastructure
Deployment and configuration-related assets.

---

## 🧠 Architecture

All tools follow:
- Safe execution principles (dry-run supported)
- Config-driven design
- Modular architecture
- Logging and observability

---

## 🚀 Goal

To build production-style cloud automation systems using Azure services and Python.
